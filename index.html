Veja abaixo todos os comandos de cada bot. Todos usam Slash Commands (/).

/sorteio criar
Cria um novo sorteio. Defina a duração (ex: 10s, 5m, 2h, 1d), número de ganhadores, prêmio e canal.
Clique no comando para copiar
/ticket criar
Abre o painel de configuração do sistema de tickets. Configure canal, cargo da staff, mensagem e imagem.
Clique no comando para copiar

Abre o painel de configuração das boas-vindas. Configure canal, mensagem e imagem de fundo.
